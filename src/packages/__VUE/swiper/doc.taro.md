# Swiper 轮播图 组件

### 介绍

常用于一组图片或卡片轮播，当内容空间不足时，可以用走马灯的形式进行收纳，进行轮播展现。

### 安装

```javascript
import { createApp } from 'vue';
import { Swiper } from '@nutui/nutui@taro';

const app = createApp();
app.use(Swiper);
```

## 代码演示

### 基础用法

`autoplay` 是否自动切换
`interval` 自动切换时间间隔
`current` 当前所在滑块的 index
`indicator-dots` 是否显示面板指示点
`indicator-color` 指示点颜色
